📝 FILE LOCATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
The .env file is now open in VS Code editor.
Fill in the two main values, then SAVE the file.

🔑 STEP 1: GET YOUR ACCOUNT ID
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: https://dash.cloudflare.com/
2. Look at the URL in your browser
3. It will look like: https://dash.cloudflare.com/abc123def456...
4. Copy everything after dash.cloudflare.com/
5. Paste it after CLOUDFLARE_ACCOUNT_ID=

Example:
CLOUDFLARE_ACCOUNT_ID=abc123def456ghi789jkl

🔑 STEP 2: CREATE API TOKEN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: https://dash.cloudflare.com/profile/api-tokens
2. Click: "Create Token" button (blue button, top right)
3. Find: "Edit Cloudflare Pages" template
4. Click: "Use template" button next to it
5. Click: "Continue to summary" (bottom)
6. Click: "Create Token" (bottom)
7. COPY the token immediately (you won't see it again!)
8. Paste it after CLOUDFLARE_API_TOKEN=

Example:
CLOUDFLARE_API_TOKEN=abc123_very_long_token_here_xyz789

💡 OPTIONAL: REUSE VALUES FOR CMS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

After filling in CLOUDFLARE_ACCOUNT_ID and CLOUDFLARE_API_TOKEN,
you can reuse them:

CF_ACCOUNT_ID=<same as CLOUDFLARE_ACCOUNT_ID>
CF_API_TOKEN=<same as CLOUDFLARE_API_TOKEN>

Or just copy-paste the same values.

📄 YOUR .ENV FILE SHOULD LOOK LIKE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CLOUDFLARE_ACCOUNT_ID=abc123your_actual_account_id
CLOUDFLARE_API_TOKEN=xyz789your_actual_long_token_here
CF_ACCOUNT_ID=abc123your_actual_account_id
CF_API_TOKEN=xyz789your_actual_long_token_here
CF_PAGES_PROJECT=jr-articles
SITE_BASE_URL=https://www.jeremyrobards.com
